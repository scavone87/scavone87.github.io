0 attività rimaste

Come funziona

Questa Todo List è stata implementata utilizzando HTML, CSS e JavaScript. Le attività vengono salvate nel localStorage del browser, quindi persisteranno anche dopo aver chiuso la pagina.

Funzionalità:

  • Aggiungere nuove attività
  • Segnare le attività come completate
  • Eliminare singole attività
  • Filtrare le attività (tutte, attive, completate)
  • Cancellare tutte le attività completate

Il codice sorgente è disponibile su GitHub.
